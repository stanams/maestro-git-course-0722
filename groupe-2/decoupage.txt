US-1 - Affichage d'une fenêtre de vérification

En tant qu'utilisateur non loggué, je veux voir une fenêtre de code de vérification apparaître une fois que j'ai validé la saisie de mon email et de mon mot de passe.

US-2 - Réception du code de vérification par sms

En tant qu'utilisateur loggué grâce à mon email, je veux recevoir un code de vérification par sms via le numéro de téléphone attaché à mon compte utilisateur.

US-3 - Saisie du code de vérification

En tant qu'utilisateur qui a reçu un code de vérification par sms, je veux pouvoir saisir ce code sur une fenêtre de vérification.

US-4 - Certification du navigateur 

En tant qu'utilisateur loggué à deux facteurs, je veux pouvoir certifier le navigateur sur lequel je me suis authentifier.
