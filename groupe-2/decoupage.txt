US-1 - Affichage d'une fenêtre de vérification


  En tant qu'utilisateur inscrit, je veux voir une fenêtre de code de vérification apparaître une fois que j'ai validé la saisie de mon email et de mon mot de passe afin de pouvoir m'authentifier avec un deuxième facteur.

      1 - En tant que user existant, je veux voir apparaître une fenêtre avec le texte suivant : "Code de vérification. La sécurité de vos informations personnelles est essentielle. Nous vérifions donc que vous êtes bien à l'origine de cette demande." afin de comprendre que je dois recevoir puis saisir un code pour me connecter.

      2 - En tant que user existant, je veux voir un champ comprenant mon numéro de téléphone grisé sur la fenêtre de vérification afin de vérifier que le numéro affiché est bien mon numéro de téléphone.

      3 - En tant que user existant, je veux voir un bouton "Recevoir un code de vérification" sur lequel je puisse cliquer sur la fenêtre de vérification afin de recevoir un code temporaire via sms.
      
      4 - En tant qu'utilisateur existant, je veux voir appraître un fenêtre de saisie du code après avoir cliqué sur le bouton "Recevoir un code de vérification".


Acceptance : Etant donné que j'ai renseigné l'email et le mot de passe attachés à un compte utilisateur actif, lorsque je clique sur le bouton de connexion, alors je vois apparaître une fenêtre d'authentification à deux facteurs.

Acceptance : Etant donné que j'ai renseigné mon numéro de téléphone lors de mon inscription, lorsque je saisis mon email et mon mot de passe pour me connecter, alors je vois apparaître une fenêtre d'authentification à deux facteurs.



US-2 - Réception du code de vérification par sms


  En tant qu'utilisateur loggué grâce à mon email, je veux recevoir un code de vérification par sms via le numéro de téléphone attaché à mon compte utilisateur afin de pouvoir m'authentifier.
  
      1 - En tant qu'utilisateur existant, je veux recevoir un code de vérification temporaire sur mon téléphone portable afin de pouvoir m'authentifier sur mon navigateur.
      
Acceptance : Etant donné que j'ai besoin de mon numéro de téléphone pour recevoir le code de vérification, lorsque j'ai entré un mauvais numéro de téléphone, alors je ne peux pas recevoir de code de vérification.
      
     
US-3 - Saisie du code de vérification


  En tant qu'utilisateur qui a reçu un code de vérification par sms, je veux pouvoir saisir ce code sur une fenêtre de vérification sur mon navigateur afin de pouvoir me connecter au site.
      
      1 - En tant qu'utilisateur qui a reçu un code de vérification par sms, je veux pouvoir saisir le code de vérification reçu par sms sur la fenêtre de saisie apparue.
      
      2 - En tant qu'utilisateur qui a reçu un code de vérification par sms, je veux voir écrit sur la fenêtre "Et pour finir... Nous venons de vous envoyer un code à 6 chiffres sur le numéro de téléphone ***. Veuillez entrer ce code pour finaliser votre connexion." afin de m'assurer que le code reçu l'a été au bon numéro.
      
      3 - En tant qu'utilisateur qui a reçu un code de vérification par sms, je veux que le numéro de télépone associé à mon compte soit en partie caché lorsqu'il est affiché sur la fenêtre de saisie du code de vérification.
      
      4 - En tant qu'utilisateur qui a reçu un code de vérification par sms, je veux voir affichées 6 cases grises afin que je puisse entrer le code de vérification que j'ai reçu par sms.
      

Acceptance : Etant donné que j'ai reçu un code de vérification par sms, lorsque je renseigne ce code sur la fenêtre de saisie à 6 chiffres, alors je suis connecté à mon compte utilisateur.  

Acceptance : Etant donné que j'ai saisi le code de vérification que j'ai reçu par sms, lorsque je saisis les chiffres dans les cases grises sur la fenêtre de connexion, alors je suis automatiquement connecté à mon compte utilisateur sans validation de la saisie.

Acceptance : Etant donné que j'ai reçu un code de vérification par sms, lorsque je rentre un code dans les cases grises qui ne correspond pas à celui reçu, je ne peux pas accéder à mon compte utilisateur.


US-4 - Certification du navigateur 


  En tant qu'utilisateur loggué à deux facteurs, je veux pouvoir certifier le navigateur sur lequel je me suis authentifié afin de ne plus passer par une authentification à deux facteurs à ma prochaine connexion.
  
      1 - En tant qu'utilisateur qui vient de s'authentifier à deux facteurs, je veux voir apparaître une fenêtre me demandant si je souhaite certifier ce navigateur via le texte "Simplifier la connexion. Si vous utilisez le navigateur de cet appareil régulièrement, le code de validation ne sera plus nécessaire sur ce navigateur." afin de comprendre que je peux éviter de recevoir un code de validation lors de ma prochaine connexion.
      
      2 - En tant qu'utilisateur qui vient de s'authentifier à deux facteurs, je veux voir apparaître un bouton "Se fier à ce navigateur" sur la fenêtre afin de certifier ce navigateur et accéder à la page d'accueil du site en tant qu'utilisateur identifié.
	  
	  3 - En tant qu'utilisateur qui vient de s'authentifier à deux facteurs, je veux voir apparaître un texte "Plus tard" sur lequel je peux cliquer afin de ne pas certifier le navigateur et accéder à la page d'accueil du site en tant qu'utilisateur identifié.
