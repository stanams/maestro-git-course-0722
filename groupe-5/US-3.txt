US-3 = saisie code verification 
 
En tant qu'utilisateur loggé qui a commencé le parcours de 2FA, 
je veux saisir sur le formulaire le code recu par sms 
afin de securiser ma session en 2FA // faire de mon navigateur un navigateur approuvé

==============================

#1 CA
ETANT DONNE QUE l'utilisateur a saisi un code valide
QUAND le 6e chiffre est saisi
ALORS l'utilisateur est redirigé vers la page "Faire confiance à ce navigateur"
      

#2 CA
ETANT DONNE QUE l'utilisateur a saisi un code invalide
QUAND le 6e chiffre est saisi
ALORS un message d'erreur apparait "Code invalide"
      ET un bouton "Renvoyer le code" apparait

#3 CA
ETANT DONNE QUE l'utilisateur a saisi un caractère qui n'est pas un chiffre
QUAND le caractère est saisi
ALORS alors un message d'erreur apparait "Seul les chiffres sont acceptés"
